How to Install

    Windows version and all-in-one executable files are under development. For now, you need to install Visual Studio Code or a similar software and to go by the following instructions to run the Program. It only works on Linux.

  • 1st Step:

    • Install a development environment like Visual Studio Code (free and easy to use) and make sure you have Installed Python on your computer. You can also do it in the Linux Terminal but it's less simple to use.

  • 2nd Step:

    • Install git on your computer or the GitHub Extension on VS Code (you don't need to create an account).

  • 3rd Step:

    • Open a new terminal and enter this command to download and open the code (click to copy):

    git clone https://github.com/paulom-s/Scope-Visual-Object-Tracking.git S-V-O-T && cd S-V-O-T && python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && code . If it doesn't works, try "python" or "py" instead of "python3" and make sure VS Code (or other) and Python are both added to the PATH.

  • 4th Step:

    • Open the "/src" folder and run "__main__.py" (arrow at the top) to launch the software, or you can try the demo programs in the "/demo" folder.
