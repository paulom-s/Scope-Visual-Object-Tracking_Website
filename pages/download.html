Scope Visual Object Tracking Software /Download

Windows version and all-in-one executable files are under development (Only works on Linux).

For now, you need to go by the following instructions to run the Program using the Terminal. You can also install a development environment like Visual Studio Code wich can be easier (download the .deb file in the VS Code Website and run "sudo apt install [path_of_the_downloaded_file].deb" in your Terminal, snap installation will not work).

Note: This tutorial is made for Ubuntu. It should work for Debian too, but I don't know for other Linux distribution.

How to Install

  • 1st Step:

    • Install / Update git on your computer with the following command (click to copy), or the GitHub Extension in VS Code (Extensions section on the left):

    sudo apt update && sudo apt install git -y

    Note: You may have to enter your password. You should see "Installing" or "Already up to date".

  • 2nd Step:

    • Still in the Linux terminal, enter this command to give the program access to the USB ports (click to copy):

    sudo usermod -a -G dialout $USER

    And then restart your computer to apply.

  • 3rd Step:

    • Install / Update Virtual Environments packages for Python with the following command (click to copy):

    sudo apt update && sudo apt install -y python3.12-venv

    Note: You may have to enter your password. You should see "Installing" or "Already up to date".

  • 4th Step:

    • Then, enter this command to download and install the code (click to copy):

    git clone https://github.com/paulom-s/Scope-Visual-Object-Tracking.git ~/Desktop/S-V-O-T && cd ~/Desktop/S-V-O-T && python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt

    If it doesn't work, make sure you have sucessfully installed git with "git --version" (it should give you a version number).

  • 5th Step:

    • If you have installed VS Code:

    Enter "code ." in the Terminal to open the Program in VS Code and then open the "/src" folder and run "__main__.py" (arrow at the top) to launch the software or try the demo programs in the "/demo" folder (see Programs's Explanation).

    If you want to run the programs in the Linux Terminal:

    To run a Program, enter "python3 [path_of_the_program].py" (see Programs's Explanation).

  • Updating:

    • The software is still in development and is updated regularly. In order to always have the latest features or patches you can run this command (click to copy) to update the program each time you use it (updates nearly every 2 days):

    cd ~/Desktop/S-V-O-T && git pull origin main

    Note: You can enter it in both the Linux and VS Code terminal, it doesn't matter.

Programs's Explanation

When you enter the S-V-O-T folder (or open it in VS Code), you will see different folders: "/.venv", "/data", "/demo", "/src" and "/tests" (this one doesn't matter). You should also have three files: ".gitignore", "README.md" and "requirements.txt".

The only things that you will use are the "/demo" and "/src" folders:

  • /demo

    • /Dependencies: Dependencies Programs for the communication with the scope or the camera. You don't need to touch it.
    • Com-Scope-Demo.py: Program to test the communication with the scope. You need to connect it. You don't need your camera.
    • Img-Analyzing-Demo.py: Program to show the functioning of the image's analyzing. You don't need your camera or your scope.
    • Program-No-Scope-Demo.py: Demo of the software without neither the camera nor the scope.
    • Program-w-Scope-Demo: Demo of the software with the scope (pre-recorded video). You don't need your camera.

  • /src

    • /Dependencies: Dependencies Programs for the communication with the scope or the camera. You don't need to touch it.
    • __main.py__: Main program, that's the one that starts the software.
