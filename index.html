About

Visual Object Tracking software for Astronomy/Astrophotography by connecting the Scope (See Compatible Mounts Below) and an DSLR Camera (See Compatible Cameras Below) to a Computer in order to visually track an object (Full-Discs or Close-Up) without Polar Alignment. It can be used for any type of Astrophotography (Planetary, Solar, Lunar, Deep-Sky, Asteroidal Occultations, ...), see the progress of the different tracking algorithms below. I'm developping it on both Windows and Linux, I am currently focusing on the Linux side (See Compatible OS in the Status and Updates page or directly on the GitHub Releases page).

I am 15, I speak both French and English, contact me for anything (See Contact Section).

My Setup

  • Scope: Celestron NexStar 127SLT (127/1500 Alt/Az GoTo)
